# Skills in CKClaw

CKClaw is a 1:1 fork of OpenClaw in terms of its skills framework. You can install any OpenClaw/Hermes skill:

```bash
ckclaw skills list                 # 52+ skills bundled + anything you install
ckclaw skills search "your query"  # browse the hub
ckclaw skills install <skill>      # install a skill
ckclaw skills inspect redteam-core # inspect a skill
```

All OpenClaw skills work. CKClaw additionally preloads `redteam-core` so red team style operations are first-class.
